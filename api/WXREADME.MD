| 参数     | 类型     | 必填  | 说明                |
|--------|--------|-----|-------------------|
| excel  | flie   | 是   | 根据excel修改企业微信个人信息 |
| domain | string | 是   | 执行excel的域账户       |

| 参数  | 说明                   |
|-----|----------------------|
| 姓名  | 修改的姓名                |
| 域账户 | 修改的域账户               |
| 属性  | 具体修改的属性，比如：志愿者、认证... |
| 内容  | 修改的内容                |
